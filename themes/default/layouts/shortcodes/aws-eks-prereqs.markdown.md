## Prerequisites

1. [Install Pulumi](/docs/install/)
2. [Install Node.js](https://nodejs.org/en/download/)
3. Install a package manager for Node.js, such as [npm](https://www.npmjs.com/get-npm) or [Yarn](https://yarnpkg.com/en/docs/install).
4. [Configure AWS credentials](/registry/packages/aws/installation-configuration/)
5. [Install AWS IAM Authenticator for Kubernetes](https://docs.aws.amazon.com/eks/latest/userguide/install-aws-iam-authenticator.html)
